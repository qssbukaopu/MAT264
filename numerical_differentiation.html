Numerical Differentiation

Forward: f'(x) ≈ [ f(x+h) − f(x) ] / h
Backward: f'(x) ≈ [ f(x) − f(x−h) ] / h
Central: f'(x) ≈ [ f(x+h) − f(x−h) ] / (2h)

Enter f(x) using JavaScript math. Examples: Math.sin(x), Math.exp(x), x**3 - 2*x, Math.log(x)

Build a degree-(n−1) Lagrange polynomial through n equally-spaced nodes in [x−h, x+h],
then differentiate analytically at x.

n nodes are placed uniformly from x−h to x+h. Larger n → higher-order polynomial. Beware Runge's phenomenon for large n.

Level 0: D(h) = [ f(x+h) − f(x−h) ] / (2h) — central difference
Level 1: R₁(h) = [ 4·D(h/2) − D(h) ] / 3
Level 2: R₂(h) = [ 16·R₁(h/4)·... ] / 15 — second Richardson level

Each Richardson level cancels one more order of truncation error. The table shows D(h), D(h/2), D(h/4) and two extrapolation levels.
